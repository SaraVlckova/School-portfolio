Introduction

This project start with basic storytelling, design character and creating sprite sheet of one part of story to learn basic, which we had to use in following group project in under this theme. Project was more focused on understanding story and dramaturgy in theory and creating in Adobe Illustrator and then end with small animation.

Task

My task was create animated character and sprite sheet. Besides that also sketched idea of story as a small game.

Challenges

My challenge was decide quick what I wanna really do and get impressive idea about story.

Gained Skills

Creating animation by implementing JavaScript on sprite sheet.

PROCESS

After collecting all ideas about character and story I would create, I decided for Buddy Bear and his story about honey. During story board I was supposed to implement Narrative curve which learn me that interesting stories have to go through exposition, raising action, climax following falling action and resolution in the end.

For designing I chose inspiration in flat design, which is popular, easy and I found it as the most common style to me.

When I was done with designing all elements of main character, I started with creating sprite sheet where I divided movement to keys, extremes and breakdowns positions, what helped me create more smooth animation than if I would me making is pose by poses.
