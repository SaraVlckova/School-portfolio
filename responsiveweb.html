Introduction

This project was based and created based on previous two projects (Interface & Mobile First). By this project I was supposed to show ability to make website responsive also for other devices as tablets and desktops.

Task

My task was edit CSS to make my first own web responsive also for other devices.

Challenges

My biggest challenge was create versions of website for tablets and desktops expanded and interesting to go through as same as mobile phone.

Gained Skills

Creating website responsive and working with bigger space.

PROCESS

After mobile version I have to developed also version for tablet and desktop, so also management of elements in the best way to keep main design principles as Gestalt principles.

By using colourful blocks on white background I kept law of figure and symmetry. By using same styling rules for elements as buttons or headings and their following management on website, I kept other laws as closure, connections, proximity or similarity.

On website I use all visual contrast except texture.

Before coding part I checked if everything fits to grids which I decided to use. Whole coding part was continuing and expanding code, which I created for project Mobile first.

Main difference between mobile and desktop version is mainly navigation bar, which is made as burger menu on smaller devices and in typical navigation for desktops.
