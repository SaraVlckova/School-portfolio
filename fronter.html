Fronter Remake
Group project

Introduction

In this project we gave more attention to change existing student app. Base on already create Proust we were supposed to come up with relevant and useful changes to improve this app for Kea Students. Base on remake and expand features we were supposed to come with new UI and learn how to correctly test it to be sure, that product is working.

Task

Our task was prototype new school app based on fronter for Kea students and then code one user flow, to prove our skills of coding we have learned until now.

Challenges

Our challenge was division of work and make code work as one.

Gained Skills

Adobe XD, Coding in more people, Work with GitHub, App Design, UX/UI

PROCESS

Our first step was analyse fronter and start collecting ideas for improvements. The biggest improvement had to be in system of features and mobile version, where fronter isn’t working in the best way.

After we collected all ideas, we start sketching our wireframes and making sticky notes about what’s gonna happen on each page.

After setting all wireframes and functions we moved to creating prototype to be able start testing our app.

Based on students answers about look, features and navigation we decided just change photos in background to more student thematic, otherwise all features and navigation worked perfectly.

While we were done with testing we moved to coding part.

Our coding part we divided through all members of our group and we started using GitHub to learn working on code in group of people on different desktops.

In coding part was challenging to keep same rules base on our agreement. For some parts of app we had to create different CSS style sheet, because otherwise it would be overwriting rules on website.

For whole UI I prepared these design guide which you can see around - logos, buttons, colour palette or typography - all important staff to be able code one app or website in group. We learned importance of this design guides while group is dividing coding like that.

All project give us big experience and knowledge how hard it can sometimes be.
