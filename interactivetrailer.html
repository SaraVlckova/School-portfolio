Interactive Trailer
Group project

Introduction

In this project we were supposed to implement and use what we have learned in previous individual project and create our first interactive product - movie teaser. Movie teaser were supposed to work as a game and contain animation.

Task

Our task was create movie teaser working as a game, which should impress user as much as he will want watch whole movie.

Challenges

Our challenge was division of work and agreement. Another challenge was definitely JavaScript.

Gained Skills

Work in group, Adobe photoshop, creating GIF, implement JavaScript

PROCESS

Our first group project started by choosing specific movie. Based on movie we create moodpboard and persona collage what we have learned from previous individual project in this theme.

Persona collage and moodboard helped us to have clear view and build our style and for who we are doing it.

Next step was agreed on whole concept of game and story. So we went through brainstorming, we pick up our best ideas and start with storyboard.

Once we were done with storyboard, we started with design of characters, scenarios and elements.

We decided to divide our team on two designers and two coders to have good working flow and equal tasks. My role in this project was designer.

In designing part we find first big challenges as division of work to keep our style. In this part we could see differences between characters and scenarios, what learn us how to divide tasks better next time.

In coding part we started with building state machine diagram to have clear view how we wanna make it work and what we need to code.

In coding part we had two main coders, but basically all of us we were working on code to understand in every detail.

With work of us all we were really satisfied and we had fun with project and at that time we also have learned new knowledges.
